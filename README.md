## Getting started


```bash
  git clone https://github.com/elvstejd/crypto-profit-calc.git
  cd crypto-profit-calc 
```

```bash
  yarn install
```
```bash
  yarn start
```

    
## Tech Stack

**Client:** React, Context API, i18next 

**Server:** Node, Express
